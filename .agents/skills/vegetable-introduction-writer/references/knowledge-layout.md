# Knowledge Layout

Use this reference when source notes are missing, sparse, or the user asks how to organize Ogawa Nouen's vegetable knowledge.

## Vegetable Note Template

Create one Markdown file per vegetable under `knowledge/vegetables/`.

Suggested fields:

- `野菜名`
- `品種`
- `栽培時期・収穫時期`
- `小川農園での育て方`
- `味・食感・香り`
- `おすすめの食べ方`
- `保存方法`
- `注意点・避けたい表現`
- `紹介文に入れたい一言`

## Style Note Template

Place style guidance under `knowledge/style/`.

Suggested fields:

- `想定読者`
- `文体`
- `使いたい語彙`
- `避けたい語彙`
- `長さの目安`
- `媒体別ルール`
- `例文`

## Minimum Useful Vegetable Facts

For a grounded introduction, try to collect at least:

- vegetable name
- taste or texture
- harvest season or freshness cue
- one recommended way to eat it
- one Ogawa Nouen-specific cultivation or handling detail
